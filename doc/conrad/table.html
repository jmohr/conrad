Represents one table in the database. The Table object exposes methods such as filter(), get(), and create() to the user. This is where you will handle most of the work of finding and making new resources in the DB.

Take this example:

db = Database('MyDSN')
artists = db['artist']
print artists.all()
[{'id':1, 'name':...}]
johnny = artists.create(name="Johnny Cash")
print johnny
{'id':7, 'name':'Johnny Cash'}
some_artists = artists.filter(id=gte(4))
print some_artists
[{'id':4,...}]

A table will often return Queries, which are made up of many lazy-loaded Resources. Some methods -- such as get() -- return a single Resource. Think of a Table as a way of finding and managing Resources.
