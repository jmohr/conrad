tables. A Database instance can be treated as a dictionary of tables. By default, the init method uses conrad.adapter.default as its adapter, which is a generic ODBC connection. You can pass in another adapter to connect to another database type.

An example:

>>> import conrad
>>> db = conrad.Database('MyDSN')
>>> print db.tables
{'album': album, 'artist': artist}
>>> print db['album'].all()
[{'artist_id':1, 'id':1, ...}]
